# Frontend - Gestión de Proyectos

Sistema de gestión de proyectos y clientes desarrollado con **Angular 21**, utilizando **PrimeNG** para componentes de UI y notificaciones globales, y **Vitest** como motor de pruebas unitarias.

---

## 🛠️ Stack Tecnológico

- **Framework:** Angular 21.2.x (Standalone Components & Signals API)
- **Diseño y UI:** PrimeNG 21.x + UI Themes (Aura Preset)
- **Manejo de Estado:** RxJS (Patrón Subject para desuscripción segura) y AuthStore para JWT
- **Testing:** Vitest + JSDOM

---

## Servidor de Desarrollo

npm start

---

## Construcción (Build)

npm run build

---

## Pruebas Unitarias

npm test
